    <h1> CRAWDAD Dataset Contribution Form </h1>
    <p>
      Thank you for offering to contribute your data to
      CRAWDAD! If we will be only mirroring data that is already hosted elsewhere, only
      your permission is needed. However, if your data will be only hosted at
      CRAWDAD, you will need to sign <a target="_blank" href="other/CRAWDAD-Contribution-Agreement.pdf">
        this agreement form.</a> Please ensure that you are able to sign the agreement before proceeding
      with submission of the description of your data. Please mail your signed agreement to
      <a target="_blank" href="mailto:crawdad-team@cs.dartmouth.edu">crawdad-team@cs.dartmouth.edu.</a>
    </p>
    <p>
      A dataset is a set of wireless network data, collected by the same organization in the course of the same research
      project. The dataset can consist of multiple tracesets, and a traceset can contain multiple traces. A traceset is
      a set of traces collected using the same measuring tool, e.g. tcpdump, snmp, syslog, or collected on the same
      date. A trace is the minimal unit of data collection, generally a file or a set of files collected by the same
      device at the same time. Here is an example of a dataset, as displayed on our website: 
      <a target="_blank" href="http://crawdad.org/ucsd/sigcomm2001/20020423/">ucsd/sigcomm2001.</a>
    </p>
    <p style="margin: 0px">
      Notes about the form:
    </p>
    <ul style="margin: 0px;">
      <li>Required Fields are denoted by an *.</li>
      <li>You may save progress at any point and return to the form using the URL emailed to the primary author.</li>
      <li>Saving progress redirects back to the "Dataset Information" first page.</li>
      <li>The 'Submit' button appears on the last traceset page.</li>
      <li>Once you submit the form, you will not be able to re-access the form via the URL. For any changes/edits, 
        please contact <a target="_blank" href="mailto:crawdad-team@cs.dartmouth.edu">crawdad-team@cs.dartmouth.edu.</a></li>
    </ul>
